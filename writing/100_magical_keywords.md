# MTG's 100 Most Magical Keywords — Topic List

100 deep-dive topics, each pairing an MTG keyword/action verb with the historical or esoteric register where the word *also* lives. Generated deterministically from Magic's evergreen + deciduous keyword lists, augmented by the game's vocabulary of magical action verbs.

Each topic is ready to feed into the `mtg-historical-trip` skill (`source_type: etymology`). Suggested approach: batch in groups of 12, since the user has previously framed batches at that size.

The list is **not** drawn from the user's tweets — `C:\Dev\SocialsDB` does not have an accessible tweet archive (the metadata indicates `twitter: 1` entry, no DB file). If a tweet archive becomes available, the list can be re-ranked by attention or sentiment.

## How to use this list

1. Pick a batch (say items 1–12: the action verbs).
2. Invoke `mtg-historical-trip` with `source_type: etymology` and pass the batch.
3. Skill writes one research card per term to `MTGSLIDER/writing/historical_trips/etymology/`.
4. Use the cards as slide thesis text or speaker notes for the matching MTGSLIDER deck.

For mechanics that don't have a dedicated deck yet, the bulk runner (`tools/bulk_generate.py`) can create one with a Scryfall query like `kw:scry` or `o:conjure`.
