# Historical Trips — Index

Research cards generated by the `mtg-historical-trip` skill. Each is a markdown file with structured YAML frontmatter (slug / source_type / title / summary / confidence / review_status / mtg_card_refs / mtg_theme_refs) plus a body containing the full scholarly content.

The frontmatter is what `tools/ingest_historical_trips.py` reads into the SQLite `historical_trips` table. The body is preserved verbatim as `body_markdown` for the overlay layer (and for human reading).
